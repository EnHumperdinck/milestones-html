Using the <em> and <i> tag.

On appearance, both tags churn out characters in italics. This is italics with the <em> tag and this italics with the <i> tag. The difference is that em stands for emphasis and i stands for italics, they basically function the same way.

Using the <b> and <strong> tag.

The same goes for the b and strong tags, both create b and strong, notice how both of them are just bolded. So b is for bold, and strong is for an even stronger emphasis. Very similar function.

Using the <code> tag.

The code tag is used to display text as if it looks like code. For example, if we were to talk about this tag, it would be written as the code tag.
