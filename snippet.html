Look at the following:

Want to search the web? Try Google!

This looks like this in html code:

<p> Try <a href="https://google.com" id="the-best-link" class="banana" target="_blank">Google</a>! </p>

Here are some resources to help you with each tag:

For <p> . Also try <p> .

For <a> . Also try <a> .

Since <a> is inside <p>, this means that <a> is a child of <p>

<p> means that it opens a paragraph. As a result, it must be closed, which is what </p> does.

<a> means that a hyperlink is opened, and it also has to be closed, which </a> does.

The attributes in the aforementioned line of code relate to <a>, and they are href, id, class, target.

The attribute href is linked to the value "https://google.com", and it is a hypertext reference, or a link to another page, the URL of such.

The attribute id is linked to the value "the-best-link", and it is a unique id for that element, useful for when we want to change certain aspects of it in the style section (css sheet)

The attribute class is linked to the value "banana", and it is also an identifier like id, but instead it allows for the creation of a class of "classmates", so you can have multiple elements inside a class and change their styles all at once on the style sheet.

The attribute target is linked with the value "blank", and target decides what happens when the hyperlink is clicked, and where the new page appears/how it will appear. The value of blank means that it is going to open up the hyperlinked page in a new window or tab.
